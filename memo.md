At Corder の練習用のディレクトリ

## 環境設定

```
Macbook Pro:
Python: 3.9
```

### コマンドまとめ

- 新しいコンテストを作成する

```
acc new [ContestName]
```

- コンテスト内で問題を追加する

```
cd [ContestName]
acc add [ContestName]
```

- 解いた問題を提出する

```
acc submit [FileName]
```
